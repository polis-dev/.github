
# My Favorite Color (color)

A feature to remind you of your favorite color

## Example Usage

```json
"features": {
    "ghcr.io/polis-dev/devc/color:1": {
        "version": "latest"
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| favorite | Choose your favorite color. | string | red |
